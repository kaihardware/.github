# kaihardware organization defaults

This special repository provides the public organization profile and default community health files for repositories that do not override them. Product-specific templates belong in the product repository.

## Contents

- profile/README.md — public organization profile
- .github/ISSUE_TEMPLATE/ — shared intake forms
- pull_request_template.md — minimum PR review contract
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md — shared policy
- CODEOWNERS — ownership placeholder requiring human assignment

Defaults are a safety net, not a reason to erase product context.
